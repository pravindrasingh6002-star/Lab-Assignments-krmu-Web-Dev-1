# Prompt Quizzer – A Console-Based JavaScript Quiz Game

This project is a simple quiz game built using core JavaScript concepts such as arrays, loops, functions, conditionals, and user input via prompt().

## Features
- 5 quiz questions stored in an array of objects.
- Prompt-based input and alert-based feedback.
- Input normalization using toLowerCase() and trim().
- Score tracking.
- Final score display at the end of the quiz.
- Runs entirely inside the browser console (no HTML required).

## How to Run
1. Open Google Chrome.
2. Open Developer Tools:
   - Windows: Ctrl + Shift + J
   - Mac: Cmd + Option + J
3. Open the Console tab.
4. Paste the code from `quiz.js`.
5. Press Enter to start the quiz.

## Files in Repository
- `quiz.js` → Contains the complete quiz program.
- `README.md` → Instructions and details about the project.

## Learning Outcomes
- Working with arrays and objects.
- Looping through questions.
- Normalizing input using string methods.
- Understanding prompt() and alert().
- Tracking score using variables.
- Using functions for clean and organized code.
